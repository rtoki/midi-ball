# midi-ball
 
技術経営論文付随プログラム
 
# Features

midiデバイス2本で画面上のボールを操作
 
# Requirement
  
* kivy 3
* rtmidi
 
# Installation
  
```bash
pip install kivy
pip install rtmidi
```
 
# Usage

parameters.txtを編集

```bash
git clone https://github.com/rtoki/midi-ball
cd midi-ball
python midi-ball.py
```
